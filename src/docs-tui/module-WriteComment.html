    WriteComment

    Data

    Name Type Default value Description
    buttonType string - 1:Comment -- 2:Reply -- 3:Edit
    content string - content entered by the user
    parentLevel integer - level of indentation of the parent comment in the view to create the current comment's level
    parentIdx integer - idx of the parent of the comment in the array of comments
    parentID string - ID of the parent component
    currentID string - ID of the the current comment
    editClicked boolean false to hide edit box after editing
    replyClicked boolean false to hide reply box after replying
